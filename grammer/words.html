Symbol

  • 横杠 - : (short)dash , minus
  • 斜杠 \: slash

Math

  • x^n :

    • x (raised) to the n-th power
    • x (raised) to the power of n
    • x to the nth

  • x^2 : x squared

  • x^3 : x cubed

  • n✓x : n-th root of x

  • 2✓x : square root of x

  • 3✓x : cube root of x

  • log(x) : logarithm of x

  • x * y = z:

    • x times y (equals/is/equal to/makes) z
    • x multiplied y makes z
    • multiply x by y, we get z

  • x / y:

    • x divided by y

  • x - y = z:

    • x minus y
    • y taken from x
    • substract y from x

  • x + y:

    • x plus y
    • add x and y
    • multple values: add them up

  • $\frac{x}{y}$ (Fractions):

    • x yths, e.g.
    • $\frac{2}{3}$ two thirds
    • $\frac{9}{10}$ nine tenths
    • 1/2: a(one) half
    • 1/3: a(one) third
    • 1/4: a quarter/ one fourth
    • 1/5: a(one) fifth

Common Word

  • agile : 敏捷灵活的
  • arbitrary : 任意的
  • crucial: 重要的，决定性的
  • declarative: 声明式
  • dedicated: 专属的
  • elastic: 灵活有弹性的,可伸缩的
  • ephemeral: 短暂性的
  • explicit: 明确的
  • granular: 颗粒的
    • fine-grained: 细粒度的
    • coarse-grained: 粗粒度的
  • amplification: 放大
    • write/read/space/request amplification
  • Descending: 降序的
  • ascending: 升序的
  • preorder traversal: 前序遍历
  • inorder traversal: 中序遍历
  • postorder traversal: 后续遍历
  • successor: 继任者,下一个, e.g. inorder successor
  • consecutive: 连续地
  • time series database: 时序数据库
  • data skew: 数据倾斜
  • consensus: n.一致
  • backward compatibility 过去的兼容
  • forward compatibility 未来的兼容
